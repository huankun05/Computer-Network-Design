# POP3 协议（邮局协议版本 3）

## 基本信息
| 属性 | 值 |
|------|-----|
| 协议名称 | POP3（Post Office Protocol version 3） |
| 所属层 | 应用层 |
| 传输层协议 | TCP |
| 默认端口 | 110（非加密）/ 995（POP3S） |
| RFC 文档 | RFC 1939 |

## 协议简介
POP3 是最简单的邮件接收协议。用户通过 POP3 客户端从邮件服务器**下载邮件到本地**，下载后邮件通常在服务器端被删除。

## 工作模式
```
邮件客户端 ──POP3──→ 邮件服务器
  (本地)               (远程)
     │                    │
     ├── USER username ──→│  登录
     ├── PASS password ──→│
     ├── STAT ───────────→│  查看邮件数量和大小
     ├── LIST ───────────→│  列出所有邮件
     ├── RETR 1 ─────────→│  下载第 1 封邮件
     ├── DELE 1 ─────────→│  删除第 1 封邮件
     ├── QUIT ───────────→│  退出（执行删除）
     │                    │
     ▼ 邮件已下载到本地     ▼ 邮件已从服务器删除
```

## POP3 命令
| 命令 | 说明 |
|------|------|
| USER | 指定用户名 |
| PASS | 指定密码 |
| STAT | 查询邮件总数和总大小 |
| LIST [msg] | 列出邮件（可选指定编号） |
| RETR msg | 下载指定编号的邮件 |
| DELE msg | 标记删除指定编号的邮件 |
| RSET | 取消所有删除标记 |
| QUIT | 退出并执行标记的删除操作 |

## 会话的三个状态
| 状态 | 说明 |
|------|------|
| 认证状态 | 用户登录、身份验证 |
| 事务状态 | 执行 STAT/LIST/RETR/DELE 等操作 |
| 更新状态 | QUIT 后执行删除并断连 |

## POP3 vs IMAP
| 特性 | POP3 | IMAP |
|------|------|------|
| 邮件存储位置 | 下载到本地 | 保留在服务器 |
| 离线访问 | 支持（已在本地） | 需同步后支持 |
| 多设备访问 | 不友好 | 天然支持 |
| 服务器存储 | 占用少 | 占用多 |
| 搜索 | 本地搜索 | 服务器搜索 |
| 文件夹管理 | 不支持 | 支持 |
| 适用场景 | 单设备简单使用 | 多设备协作 |

## 适用场景
- 用户习惯在单台设备上管理邮件
- 需要节约邮件服务器存储空间
- 需要离线查看历史邮件
