# 传输层 — 知识文档索引

传输层是 TCP/IP 五层模型的第四层，负责提供端到端的数据传输服务。

## 本层主要功能
- 提供进程到进程的通信（端口寻址）
- 分段和重组数据
- 流量控制和拥塞控制
- 差错检测和恢复

## 本层核心协议
| 协议 | 特点 | 适用场景 |
|------|------|---------|
| TCP | 面向连接、可靠、字节流 | Web浏览、文件传输、邮件 |
| UDP | 无连接、不可靠、数据报 | 视频流、DNS查询、游戏 |
