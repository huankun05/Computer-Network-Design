# 网络层 — 知识文档索引

网络层是 TCP/IP 五层模型的第三层，负责将数据包从源主机传送到目的主机。

## 本层主要功能
- 逻辑寻址（IP 地址）
- 路由选择（确定最佳路径）
- 分组转发（将数据包从源传送到目的）
- 拥塞控制

## 本层核心协议
| 协议 | 说明 |
|------|------|
| IP | 网际协议，核心路由协议 |
| ICMP | 互联网控制消息协议 |
| ARP | 地址解析协议（IP → MAC） |
| RARP | 反向地址解析协议（MAC → IP） |
| OSPF | 开放最短路径优先（内部网关协议） |
| RIP | 路由信息协议（内部网关协议） |
| BGP | 边界网关协议（外部网关协议） |

## 本层设备
| 设备 | 说明 |
|------|------|
| 路由器 | 连接不同网络，根据 IP 地址转发数据包 |
| 三层交换机 | 具有路由功能的交换机 |
