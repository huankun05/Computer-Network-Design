# 数据链路层 — 知识文档索引

数据链路层是 TCP/IP 五层模型的第二层，负责在相邻节点之间可靠地传输数据帧。

## 本层主要功能
- 帧的封装和解封装
- 物理地址寻址（MAC 地址）
- 差错检测（CRC）
- 流量控制
- 介质访问控制

## 本层核心协议
| 协议 | 说明 |
|------|------|
| 以太网 (Ethernet) | 最常用的局域网技术 |
| PPP | 点对点协议 |
| HDLC | 高级数据链路控制 |

## 本层设备
| 设备 | 说明 |
|------|------|
| 交换机 | 根据 MAC 地址表进行帧转发 |
| 网桥 | 连接两个局域网段 |
