# 物理层 — 知识文档索引

物理层是 TCP/IP 五层模型的最底层，负责在物理介质上传输原始比特流。

## 本层主要功能
- 比特流的传输
- 电气、机械、功能和规程特性定义
- 信号编码和调制

## 传输介质
| 介质 | 特点 | 速率 | 距离 |
|------|------|------|------|
| 双绞线 | 最常用，成本低 | 1Gbps-10Gbps | 100m |
| 同轴电缆 | 抗干扰强 | 10Mbps-10Gbps | 500m |
| 光纤 | 带宽极高，抗干扰 | 10Gbps-100Gbps | 数十km |
| 无线电磁波 | 移动性强 | 11Mbps-Gbps | 数十-数百m |

## 本层设备
| 设备 | 说明 |
|------|------|
| 集线器 (Hub) | 将信号广播到所有端口 |
| 中继器 (Repeater) | 放大和整形信号，延长传输距离 |
