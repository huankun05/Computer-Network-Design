# 中继器（Repeater）

## 基本信息
| 属性 | 值 |
|------|-----|
| 设备名称 | 中继器（Repeater） |
| 所属层 | 物理层 |
| 工作方式 | 信号放大和整形 |
| 端口数 | 通常为 2 口 |

## 信号衰减问题
信号在传输介质中传播时，随距离增加会逐渐衰减和失真：

```
发送端 ────────────────────── 接收端
  |        信号衰减→           |
 (强)                        (弱)
```

距离超过最大传输距离后，接收端无法正确识别信号。此时需要中继器。

## 工作原理
1. 中继器接收衰减和失真的信号
2. 对信号进行**放大**
3. 对信号波形进行**整形**（恢复标准波形）
4. 重新生成干净的信号后转发出去
5. 有效延长信号传输距离

## 关键特点
- **不理解帧结构**：不解析 MAC 地址、IP 地址等任何协议字段
- **不做过滤**：所有信号（包括噪声和错误帧）都放大转发
- **不隔离冲突域**：两边属于同一个冲突域
- **不能连接不同类型网络**：只能连接相同介质的网段

## 中继器 vs 交换机 vs 路由器
| 特性 | 中继器 | 交换机 | 路由器 |
|------|--------|--------|--------|
| 工作层 | 物理层 | 数据链路层 | 网络层 |
| 处理依据 | 电信号 | MAC 地址 | IP 地址 |
| 冲突域 | 不隔离 | 隔离 | 隔离 |
| 广播域 | 不隔离 | 不隔离 | 隔离 |

## 5-4-3 规则（经典以太网）
- 任意两个节点之间最多 **5 个网段**
- 最多 **4 个中继器**
- 其中最多 **3 个**网段可以连接主机（其余为链路段）

## 现状
- 传统中继器已被交换机和路由器取代
- 但中继器概念仍然存在：Wi-Fi 中继器、光纤中继器等扩展了传输距离
