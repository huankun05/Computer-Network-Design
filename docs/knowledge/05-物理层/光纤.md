# 光纤（Fiber Optic）

## 基本原理
光纤利用**光的全反射**原理传输数据。光信号在纤芯中通过全反射不断向前传播，几乎无损耗。

```
光信号 ──→  ~~/\/\/\/\/\/\~~ ──→ 光信号
  │           全反射传播        │
(发送端)                       (接收端)
```

## 分类
| 类型 | 光源 | 纤芯直径 | 传输距离 | 典型用途 |
|------|------|---------|---------|---------|
| 单模光纤 (SMF) | 激光 | 9 μm | 数十 km 以上 | 城域网、长途干线 |
| 多模光纤 (MMF) | LED | 50/62.5 μm | 数百 m ~ 2 km | 园区网、数据中心 |

### 为什么单模传输更远？
- 单模：只有一条光路传输，无色散，衰减小
- 多模：多条光路同时传输，模式色散导致信号失真，距离受限

## 光纤结构
```
        涂覆层 (250 μm)
      ┌─────────────────┐
      │  包层 (125 μm)   │
      │  ┌───────────┐  │
      │  │纤芯(9/50μm)│  │
      │  └───────────┘  │
      └─────────────────┘
```

## 光纤 vs 双绞线
| 特性 | 光纤 | 双绞线 |
|------|------|--------|
| 带宽 | 极高（Tbps 级） | 最高 40 Gbps |
| 距离 | 数十 km | 100 m |
| 抗电磁干扰 | 完全免疫 | 受干扰 |
| 安全性 | 极高（难以窃听） | 较低 |
| 成本 | 较高 | 低 |
| 安装难度 | 需专业设备 | 简单 |
| 重量 | 轻 | 较重 |

## 光纤连接器类型
| 类型 | 特点 |
|------|------|
| SC | 方形，推拉式，常见于交换机 |
| LC | 小型，常用于 SFP 模块 |
| ST | 圆形卡口式 |
| FC | 螺纹连接，高稳定性 |

## 应用场景
- 网络主干（骨干网）
- 海底光缆（跨洋通信）
- FTTH（光纤到户）
- 数据中心内部互联
